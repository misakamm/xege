## 自己编译ege源代码

从上方源代码页面获取源代码, 然后根据自己需要编译

>关于32位和64位：注意不是你的操作系统是多少位就下载多少位，而是你的编译器是多少位就用多少位，或者能支持多少位就用多少位。
>如果是MinGW，那么请用32位的，如果是vc9/vc10，那可以两个都下载。如果你不知道，两个都下载就没错了，
不过，目前多数情况下都是32位。
