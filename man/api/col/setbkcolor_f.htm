主页 / 库函数目录 / 颜色 / setbkcolor_f

功能：

设置窗口（图像）的背景颜色。
仅设置 cleardevice() 所使用的颜色，不会替换图像中的像素颜色。
调用 setbkcolor_f() 后再调用 cleardevice()  即可将整个图像填充成新的背景颜色。

声明：

void setbkcolor_f(
    color_t color,
    PIMAGE pimg = NULL
);


参数：

color
指定要设置的背景颜色。注意，该设置会同时影响文字背景色。

pimg
要设置背景颜色的图像（NULL 为窗口）。

返回值：

（无）
