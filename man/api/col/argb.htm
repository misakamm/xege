主页-＞库函数目录-＞颜色表示及相关函数-＞EGEARGB

功能：
EGEARGB 宏用于通过透明度和红、绿、蓝颜色分量合成颜色。

声明：

color_t EGERGBA(
    BYTE byAlpha,   // 颜色的透明度
    BYTE byRed,     // 颜色的红色部分
    BYTE byGreen,   // 颜色的绿色部分
    BYTE byBlue     // 颜色的蓝色部分   
    );



参数：
byAlpha
颜色的透明度，取值范围：0~255，0表示完全透明，255表示不透明。
byRed
颜色的红色部分，取值范围：0~255。
byGreen
颜色的绿色部分，取值范围：0~255。
byBlue
颜色的蓝色部分，取值范围：0~255。

返回值：
返回合成的颜色。
