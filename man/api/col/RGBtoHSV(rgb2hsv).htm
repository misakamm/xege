主页 / 库函数目录 / 颜色 / RGBtoHSV(rgb2hsv)

功能：

将 RGB 颜色转换为 HSV 颜色。

声明：

void rgb2hsv(
    color_t rgb,
    float *H,
    float *S,
    float *V
);

#define RGBtoHSV rgb2hsv


参数：
rgb
原 RGB 颜色。

H
原 HSL 颜色模型的 Hue(色相) 分量，0.0 ≤ H ＜ 360.0。

S
原 HSL 颜色模型的 Saturation(饱和度) 分量，0.0 ≤ S ≤ 1.0。

V
原 HSL 颜色模型的 Value(明度) 分量，0.0 ≤ L ≤ 1.0。

返回值：

（无）

说明：
HSV 详见 hsv2rgb。
