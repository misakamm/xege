主页 / 库函数目录 / 颜色 / RGBtoHSL(rgb2hsl)

功能：

将 RGB 颜色转换为 HSL 颜色。

声明：

void rgb2hsl(
    color_t rgb,
    float *H,
    float *S,
    float *L
);

#define RGBtoHSL rgb2hsl


参数：

rgb
原 RGB 颜色。

H
原 HSL 颜色模型的 Hue(色相) 分量，0.0 ≤ H ＜ 360.0。

S
原 HSL 颜色模型的 Saturation(饱和度) 分量，0.0 ≤ S ≤ 1.0。

L
原 HSL 颜色模型的 Lightness(亮度) 分量，0.0 ≤ L ≤ 1.0。

返回值：

（无）

说明：

HSL 详见 hsl2rgb。
