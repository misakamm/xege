功能：

设置输出文字时的背景模式。

声明：

void setbkmode(
    int bkMode,
    PIMAGE pimg = NULL
);


参数：

bkMode
指定输出文字时的背景模式，可以是以下值：



	
值			描述

	
OPAQUE（默认）		文字输出时带着背景色块。背景色块的颜色可用 setfontbkcolor(), setbkcolor() 设置。

	
TRANSPARENT	文字输出时无背景色块	



pimg
要设置文字背景模式的图像（NULL 为窗口）。

返回值：

（无）

（无）
