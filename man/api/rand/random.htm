主页 / 库函数目录 / 随机数 / random

功能：

生成某范围内的随机整数

声明：

unsigned int random(unsigned int n = 0);

参数：
n
生成 0 到 n-1 范围内的整数。
如果 n 为 0，则返回 0 到 0xFFFFFFFF 范围内的整数。

返回值：

一个随机整数。

其它说明：

建议使用本函数生成随机数而不是使用 <stdlib> 里的 rand() 函数。因为本函数使用专业的随机数生成算法，随机性远超系统的rand函数。
另，千万不要自己使用 random() % n 的方式取获得一个范围内的随机数，请使用random(n)，切记。本随机序列的初始化只能调用 randomize() 函数，不能使用 srand()。
