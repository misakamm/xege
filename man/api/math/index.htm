VECTOR3D					库提供的三维向量类，以下为类的成员简介

VECTOR3D::operator -		三维向量减法

VECTOR3D::operator &		向量叉乘											


VECTOR3D::GetAngel			计算两个三维向量的夹角								


VECTOR3D::GetModule		计算三维向量的模									


VECTOR3D::GetSqrModule>	计算三维向量模的平方								


VECTOR3D::Rotate			三维向量绕另一任意向量旋转，或者按指定旋转角旋转	


VECTOR3D::SetModule		在保持方向不变的情况下把三维向量长度改为设定值
