主页 / 库函数目录 / 文字输出 / rectprintf

功能：

在指定矩形区域内格式化输出字符串。

声明：

void rectprintf(
    int x,
    int y,
    int w,
    int h,
    LPCSTR format,
    ...
);

void rectprintf(
    int x,
    int y,
    int w,
    int h,
    LPCWSTR format,
    ...
);


参数：
x, y, w, h
要输出字符串所在的矩形区域

format
格式字符串。

...
要输出的内容的参数，类似 printf。

返回值：

（无）
