主页-＞库函数目录-＞文字输出相关函数-＞xyprintf

功能：
这个函数用于在指定位置格式化输出字符串。

声明：

void xyprintf(
    int x,
    int y,
    LPCSTR textstring,
    ...
);
void xyprintf(
    int x,
    int y,
    LPCWSTR textstring,
    ...
);



参数：
x
字符串输出时头字母的 x 轴的坐标值
y
字符串输出时头字母的 y 轴的坐标值。
textstring
要输出的字符串的指针。
...
要输出的内容的参数，类似printf。

返回值：
（无）
