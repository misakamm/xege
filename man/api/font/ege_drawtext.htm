主页 / 库函数目录 / 文字输出 / ege_drawtext

功能：

在指定位置格式化输出字符串。文本颜色使用 ARGB 颜色，由 setcolor() 指定。

声明：

void ege_drawtext(
    LPCSTR format,
    int x,
    int y,
    ...
);
void ege_drawtext(
    LPCWSTR format,
    int x,
    int y,
    ...
);


参数：

format
格式字符串。

x
字符串输出时头字母的 x 轴的坐标值

y
字符串输出时头字母的 y 轴的坐标值。

...
要输出的内容的参数，类似 printf。

返回值：

（无）
