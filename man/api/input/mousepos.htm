主页 / 库函数目录 / 键盘鼠标输入 / mousepos

功能：

获取当前鼠标坐标。

声明：

int mousepos(int *x, int *y);

参数：
x
用来接收横坐标

y
用来接收纵坐标

返回值：

0

示例：

if (keystate(key_mouse_l)) { // 鼠标左键按下了
    //获取鼠标当前位置坐标，放在变量mouse_x和mouse_y中 
    int mouse_x,mouse_y;
    mousepos(&mouse_x,&mouse_y);
    //在鼠标当前位置画一个半径为100的圆
    circle(&mouse_x,&mouse_y,100);
}
