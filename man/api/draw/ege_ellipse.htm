主页-＞库函数目录-＞绘制图形相关函数-＞ege_ellipse

功能：
这个函数用于画椭圆线框。边线颜色由setcolor函数决定。

声明：

void ege_ellipse(
    float x,
    float y,
    float w,
    float h,
    PIMAGE pimg = NULL
);

注意事项：
使用的是ARGB颜色，透明度不能为0，否则无绘制效果。

参数：
x
包围椭圆的矩形的左上角 x 坐标。

y
包围椭圆的矩形的左上角 y 坐标。

w
包围椭圆的矩形的宽。

h
包围椭圆的矩形的高。

pimg
绘制的目标图像，默认为NULL，即绘制到窗口。

返回值：
（无）
