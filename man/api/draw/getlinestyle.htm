功能：

获取当前线形。

声明：

void getlinestyle(
    int            *plinestyle,
    unsigned short *pupattern = NULL,
    in             *pthickness = NULL,
    PCIMAGE         pimg = NULL
);


参数：
plinestyle
返回当前线型。详见 setlinestyle。

pupattern
返回当前自定义线形数据。

pthickness
返回当前线形宽度。

pimg
目标图像，默认值为 NULL（窗口）

返回值：

（无）
