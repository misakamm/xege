功能：

绘制填充的椭圆。
边线颜色由 setcolor() 函数指定，填充颜色由 setfillcolor() 函数指定。

声明：

void fillellipse(
    int x,
    int y,
    int xradius,
    int yradius,
    PIMAGE pimg = NULL
);
void fillellipsef(
    float x,
    float y,
    float xradius,
    float yradius,
    PIMAGE pimg = NULL
);


参数：
x
椭圆的圆心 x 坐标。

y
椭圆的圆心 y 坐标。

xradius
椭圆的 x 轴半径。

yradius
椭圆的 y 轴半径。

pimg
绘制的目标图像，默认为 NULL（即绘制到窗口）。

返回值：

（无）
