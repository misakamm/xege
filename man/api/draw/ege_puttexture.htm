主页-＞库函数目录-＞绘制图形相关函数-＞ege_puttexture

功能：
在目标区域上绘制纹理图像，图像需要先用ege_gentexture()生成纹理图像。

声明：

void ege_puttexture(
    PIMAGE srcimg,           //纹理图像
    ege_rect dest,           //绘制对象区域
    PIMAGE pimg = NULL       //绘制对象
);

void ege_puttexture(
    PIMAGE srcimg,           //纹理图像
    ege_rect dest,           //绘制对象区域
    ege_rect src,            //纹理图像区域
    PIMAGE pimg = NULL       //绘制对象
);



参数：

srcimg
纹理图像

dest
绘制对象区域

src
纹理图像区域

pimg 
绘制对象，为NULL则表示设置的是窗口。

返回值：
（无）

（无）
