主页 / 库函数目录 / 图形绘制 / ege_puttexture

功能：

在目标区域上绘制纹理图像，源图像需要先用 ege_gentexture() 生成纹理图像。

声明：

void ege_puttexture(
    PIMAGE srcimg,           //纹理图像
    ege_rect dest,           //绘制对象区域
    PIMAGE pimg = NULL       //绘制对象
);

void ege_puttexture(
    PIMAGE srcimg,           //纹理图像
    ege_rect dest,           //绘制对象区域
    ege_rect src,            //纹理图像区域
    PIMAGE pimg = NULL       //绘制对象
);


参数：

srcimg
纹理图像（由 ege_gentexture() 生成)

dest
绘制对象区域

src
纹理图像区域

pimg 
绘制的目标图像，默认为 NULL（即绘制到窗口）。

返回值：

（无）
