主页 / 库函数目录 / 图形绘制 / ellipsegradient

功能：

设置高级函数颜色填充模式为椭圆渐变颜色填充模式。
有效区域为椭圆区域，区域内容颜色渐变填充。
填充区域在有效区域内才能正确填充。

声明：

void ellipsegradient(
    ege_point center,            //中心点坐标
    color_t centercolor,         //中心点颜色
    float x,
    float y,
    float w,
    float h,
    color_t color,               //边界颜色
    PIMAGE pimg = NULL
);


参数：

center，centercolor
中心点的坐标及颜色

x, y, w, h
包围椭圆的矩形的左上角坐标及宽高。

pimg
设置的目标图像，默认为 NULL，即设置窗口。

返回值：

（无）
