主页-＞库函数目录-＞绘制图形相关函数-＞pieslice

功能：
这个函数用于画填充圆扇形。

声明：

void pieslice(
    int x,
    int y,
    int stangle,
    int endangle,
    int radius,
    PIMAGE pimg = NULL
);
void pieslicef(
    float x,
    float y,
    float stangle,
    float endangle,
    float radius,
    PIMAGE pimg = NULL
);



参数：
x
圆扇形的圆心 x 坐标。

y
圆扇形的圆心 y 坐标。

stangle
圆扇形的起始角的角度。

endangle
圆扇形的终止角的角度。

radius
圆扇形的半径。

返回值：
（无）

（无）
