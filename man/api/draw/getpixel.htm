功能：

获取图像上某个像素点的颜色 (ARGB)。

声明：

color_t getpixel(
    int x,
    int y
    PCIMAGE pimg = NULL
);


参数：
x
要获取颜色的 x 坐标。

y
要获取颜色的 y 坐标。

pimg
目标图像，默认值为 NULL（窗口）

返回值：

指定点的颜色。

示例：

（无）

其它说明：另有高速版的getpixel_f函数，参数一样，作用一样，但不进行相对坐标变换和边界检查（如果越界绘图，要么画错地方，要么程序结果莫名其妙，甚至直接崩溃），并且必须在批量绘图模式下才能使用，否则将发生不可预知的结果。
