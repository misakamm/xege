主页-＞库函数目录-＞绘制图形相关函数-＞lineto

功能：
这个函数用于画线。

声明：

void lineto(
    int x,
    int y,
    PIMAGE pimg = NULL
);



参数：
x
从“当前点”开始画线，终点横坐标为 x （终点本身画不到）。

y
从“当前点”开始画线，终点纵坐标为 y （终点本身画不到）。

返回值：
（无）

示例：
（无）

其它说明：另有高速版的lineto_f函数，参数一样，作用一样，但不进行相对坐标变换和边界检查（如果越界绘图，要么画错地方，要么程序结果莫名其妙，甚至直接崩溃），并且必须在窗口锁定绘图模式下才能使用，否则将发生不可预知的结果。
