主页 / 库函数目录 / 图形绘制 / drawlines

功能：

绘制多条线段，线条颜色由 setcolor() 函数指定。

声明：

void drawlines(
    int numliness,
    const int *polypoints,
    PIMAGE pimg = NULL
);


参数：
numlines
线段数目。

polypoints
每个点的坐标（每连续两个值依次为点的 x 坐标和 y 坐标）, 数组元素个数为 numlines * 4。
每两个点画一线段。

pimg
绘制的目标图像，默认为 NULL（即绘制到窗口）。

返回值：

（无）

（无）
