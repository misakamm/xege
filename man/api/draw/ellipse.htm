主页-＞库函数目录-＞绘制图形相关函数-＞ellipse

功能：
这个函数用于画椭圆弧线。边线颜色由setcolor函数决定

声明：

void ellipse(
    int x,
    int y,
    int stangle,
    int endangle,
    int xradius,
    int yradius,
    PIMAGE pimg = NULL
);
void ellipsef(
    float x,
    float y,
    float stangle,
    float endangle,
    float xradius,
    float yradius,
    PIMAGE pimg = NULL
);



参数：
x
椭圆弧线的圆心 x 坐标。

y
椭圆弧线的圆心 y 坐标。

stangle
椭圆弧线的起始角的角度。

endangle
椭圆弧线的终止角的角度。

xradius
椭圆弧线的 x 轴半径。

yradius
椭圆弧线的 y 轴半径。

返回值：
（无）
