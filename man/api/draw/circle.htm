主页 / 库函数目录 / 图形绘制 / circle

功能：

绘制圆，此圆是空心的，不填充，边线颜色由 setcolor() 函数指定

声明：

void circle(
    int x,
    int y,
    int radius,
    PIMAGE pimg = NULL
);
void circlef(
    float x,
    float y,
    float radius,
    PIMAGE pimg = NULL
);


参数：
x
圆的圆心 x 坐标。

y
圆的圆心 y 坐标。

radius
圆的半径。

pimg
绘制的目标图像，默认为 NULL（即绘制到窗口）。

返回值：

（无）
