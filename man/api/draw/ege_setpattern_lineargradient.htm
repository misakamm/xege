主页-＞库函数目录-＞绘制图形相关函数-＞ege_setpattern_lineargradient

功能：
设置高级函数颜色填充模式为线性渐变颜色填充模式。
区域为从以直线为宽的条状区域，颜色由一边向另一边线性渐变。
填充区域在区域内有效。


声明：

void ege_setpattern_lineargradient(
    float x1,
    float y1,
    color_t c1,
    float x2,
    float y2,
    color_t c2,
    PIMAGE pimg = NULL
);



参数：
x1, y1, c1
第一个点的坐标及颜色

x2, y2, c2
第二个点的坐标及颜色

pimg
设置的目标图像，默认为NULL，即设置窗口。

返回值：
（无）

（无）
