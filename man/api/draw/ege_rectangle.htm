功能：

绘制矩形线框，线框颜色由 setcolor() 指定。

声明：

void ege_rectangle(
    float x,
    float y,
    float w,
    float h,
    PIMAGE pimg = NULL
);



注意事项：
使用的是 ARGB 颜色，透明度不能为 0，否则无绘制效果。

参数：
x
矩形左上角 x 坐标。

y
矩形左上角 y 坐标。

w
矩形宽。

h
矩形高。

pimg
绘制的目标图像，默认为 NULL（即绘制到窗口）。

返回值：

（无）
