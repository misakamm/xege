主页-＞库函数目录-＞绘制图形相关函数-＞putpixels

功能：
这个函数用于画多个点。

声明：

void putpixels(
    int nPoint,
    int* pPoints,
    PIMAGE pimg = NULL
);



参数：
nPoint
点的数目。

pPoints
指向点的描述的指针，一个int型的数组，依次每三个int描述一个点：第一个为x坐标，第二个为y坐标，第三个为颜色值。

返回值：
（无）

示例：
（无）

其它说明：另有高速版的putpixels_f函数，参数一样，作用一样，但不进行相对坐标变换和边界检查（如果越界绘图，要么画错地方，要么程序结果莫名其妙，甚至直接崩溃），并且必须在窗口锁定绘图模式下才能使用，否则将发生不可预知的结果。
