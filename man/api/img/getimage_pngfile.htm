主页-＞库函数目录-＞图像处理相关函数-＞getimage_pngfile

功能：
这个函数用于从png图片文件中获取图像。

声明：

int getimage_pngfile(
    PIMAGE  pimg,
    LPCSTR  filename
);

int getimage_pngfile(
    PIMAGE  pimg,
    LPCWSTR  filename
);



参数：
pimg
保存图像的PIMAGE，需要先使用newimage()创建。获取后图像大小与原图大小一致，而不是图像原先的大小

filename
图像文件名


返回值：
成功返回0，否则返回非0，若pimg传入错误，会引发运行时异常。
