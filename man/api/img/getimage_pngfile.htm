主页 / 库函数目录 / 图像处理 / getimage_pngfile

功能：

从 PNG 图片文件中获取图像。获取后图像大小与原图大小一致，而不是图像原先的大小。

声明：

int getimage_pngfile(
    PIMAGE  pimg,
    LPCSTR  filename
);

int getimage_pngfile(
    PIMAGE  pimg,
    LPCWSTR  filename
);


参数：
pimg
图像对象的指针，需要先使用 newimage() 创建图像。

filename
图像文件名


返回值：

成功返回 0，否则返回非 0，若 pimg 传入错误，会引发运行时异常。
