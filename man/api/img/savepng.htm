功能：

将图像保存到文件中。

声明：

int savepng(
    PCIMAGE  pimg,
    LPCSTR  filename,
    int  bAlpha = 0
);

int savepng(
    PCIMAGE  pimg,
    LPCWSTR  filename,
    int  bAlpha = 0
);


参数：
pimg
要保存的图像

filename
保存的图像文件名

bAlpha
设置保存图像的透明度，范围从0 到 255

返回值：

成功返回0，否则返回非0，若pimg传入错误，会引发运行时异常。
